### Linguagens e Frameworks Utilizados no Projeto

Este projeto de landing page para o time de futebol utiliza as seguintes tecnologias:

## Linguagens de Programação

- **TypeScript**: Utilizado como linguagem principal, oferecendo tipagem estática para JavaScript, como visto nas extensões `.tsx` dos arquivos
- **HTML**: Incorporado dentro da sintaxe JSX/TSX
- **CSS**: Aplicado através das classes do Tailwind CSS


## Frameworks e Bibliotecas

- **React**: Framework JavaScript para construção da interface de usuário e componentes
- **Next.js**: Framework React para desenvolvimento web, utilizando a arquitetura App Router
- **Tailwind CSS**: Framework CSS utilitário para estilização rápida e responsiva
- **shadcn/ui**: Biblioteca de componentes UI baseada no Radix UI (usado para o componente Button)
- **Lucide React**: Biblioteca de ícones SVG para React


## Características Técnicas

- **Componentes Reutilizáveis**: Como MatchCard, MiniCalendar e SocialLinks
- **Design Responsivo**: Implementado com classes responsivas do Tailwind (md:, lg:, etc.)
- **Estado do Cliente**: Gerenciamento de estado com React Hooks (useState no componente MiniCalendar)
- **Componentes do Lado do Servidor**: Arquitetura baseada no App Router do Next.js
- **Interatividade**: Implementada no calendário com navegação entre meses


O projeto segue uma arquitetura moderna de desenvolvimento web, com foco em componentes reutilizáveis, estilização eficiente e uma experiência de usuário responsiva e interativa.
